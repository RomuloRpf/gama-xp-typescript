# Typescript com yarn

### 1. yarn init -y
### 2. yarn add typescript -D
### 3. yarn tsc --init
### 4. yarn tsc (para converter todos os ts para js)
## __OU__
### 4. No package.json adicionar atributo scripts
### 5. Dentro de scripts criar um atributo "dev": "ts-node-dev caminho do arquivo .ts"
### 6. yarn dev
